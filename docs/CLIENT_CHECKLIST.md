# Client Checklist - Spike E-commerce

## ✅ Información Requerida del Cliente

### 1. Credenciales de Stripe

**Dónde obtenerlas:** https://dashboard.stripe.com/apikeys

Necesito:
- [ ] **Publishable Key** (empieza con `pk_test_` para pruebas)
- [ ] **Secret Key** (empieza con `sk_test_` para pruebas)
- [ ] **Webhook Secret** (lo configuramos después del deploy)

### 2. Credenciales de Mercado Pago

**Dónde obtenerlas:** https://www.mercadopago.com.mx/developers/panel/app

Necesito:
- [ ] **Public Key** (empieza con `TEST-` para pruebas)
- [ ] **Access Token** (empieza con `TEST-` para pruebas)

### 3. Catálogo de Productos

Para cada producto, necesito:
- [ ] **Nombre** del producto
- [ ] **Descripción** corta (1-2 líneas)
- [ ] **Descripción** larga (detallada)
- [ ] **Precio** en MXN (ejemplo: $500.00)
- [ ] **Imágenes** (mínimo 1, ideal 3-5 por producto)
  - Formato: JPG o PNG
  - Tamaño mínimo: 800x800 píxeles
  - Fondo: Preferiblemente blanco o transparente
- [ ] **Categoría** (opcional)
- [ ] **Stock** disponible (opcional)

**Formato de entrega de productos:**
Puede ser Excel, Google Sheets o lista simple con toda la información.

### 4. Información del Negocio

- [ ] **Nombre del negocio**
- [ ] **Logo** (PNG con fondo transparente, idealmente)
- [ ] **Colores de marca** (ejemplo: #FF5733, #3498DB)
- [ ] **Email de contacto**
- [ ] **Teléfono** (opcional)
- [ ] **Dirección física** (si aplica)
- [ ] **Redes sociales** (Facebook, Instagram, etc.)

### 5. Configuración de Envío (si aplica)

- [ ] ¿Ofrecen envío a domicilio?
- [ ] ¿Cuánto cuesta el envío?
- [ ] ¿A qué ciudades/estados envían?
- [ ] ¿Tiempo estimado de entrega?
- [ ] ¿Ofrecen pickup en tienda?

## ❓ Preguntas Frecuentes

**P: ¿Puedo empezar con pocos productos?**
R: Sí, puedes empezar con 1-5 productos y agregar más después.

**P: ¿Las credenciales de prueba son seguras?**
R: Sí, las credenciales de prueba (`test`, `TEST-`) no procesan pagos reales.

**P: ¿Cuánto cuesta Stripe/Mercado Pago?**
R: Ambos cobran comisión por transacción (≈3.6% + $3 MXN). No hay costo mensual.

**P: ¿Puedo cambiar precios después?**
R: Sí, puedes actualizar precios cuando quieras.

**P: ¿Las imágenes deben ser profesionales?**
R: No necesariamente, pero deben ser claras y mostrar bien el producto.
